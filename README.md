# TCPOnlineGameServer
WINDOWS IOCP多线程TCP游戏联机服务器（单）
===========================================================================

特点：

使用iocp高效处理并发数据

使用消费者 生产者模式（io线程只负责收取与简单处理数据（如解密）不负责处理业务逻辑，将简单处理后数据推送给业务队列中，业务线程将从队列中读取并处理数据）

使用无锁高效的多线程安全队列

使用hashmap进行客户端玩家信息保存，查找读取客户端信息很高效（使用读写锁来保障数据安全处理）

数据进行简单加密传输

可自定义数据协议   （默认按照 ：头-大小-数据）

利用信号量唤醒业务线程 避免一直循环
